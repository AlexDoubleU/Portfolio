Aston Martin DB5 Texturing

Project Type University Project
Duration 1 Month
Software Used Maya, Substance Painter & Designer

About

For this project we were given a 3D model of an Aston Martin DB5 and we had to prepare the model for texturing, texture it, and light it in a game engine.

Preparing the model for texturing involved fixing any mesh problems, creating the uvs, and setting material ids with colour coding.

Texturing involved creating the materials required for the car. I used Substance Designer to create wood, leather and the car paint. I then used Substance Painter to apply the materials to the UVs.

Once texturing was done, we had to import it into unreal engine and then properly use lighting to showcase the car.
